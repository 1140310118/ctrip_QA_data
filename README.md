# 爬取携程问答数据

实现方法
- 使用requests下载网页
- 使用BeautifulSoup解析网页
- 使用threading实现多线程

起始地址：[http://you.ctrip.com/asks/topics.html](http://you.ctrip.com/asks/topics.html)

- 从专题页获取标签地址
- 从标签页获取问题地址
- 从问题地址爬取问答数据

共爬取问答数据132MB（压缩后）。若改进爬取方法，应该能爬取更多数据。
